# FMK Gulve - Portfolio Demo Website

🚧 **This is a portfolio demonstration project** 🚧

A modern, multilingual flooring company website built with Next.js 15, showcasing professional web development skills and best practices.

## 🌐 Live Demo

**Website:** [https://fmkgulve.vercel.app/da](https://fmkgulve.vercel.app/da)

## 📋 Project Overview

FMK Gulve was originally developed as a fully functional business website for a Danish flooring company. After the company ceased operations, this project has been converted into a portfolio demonstration piece that showcases:

- **Modern Web Development** - Built with the latest Next.js 15 and React 19
- **Multilingual Support** - Full Danish/English internationalization
- **Professional Design** - Clean, responsive UI with Tailwind CSS
- **Contact Forms** - Demo functionality (no actual emails sent)
- **SEO Optimized** - Proper meta tags and static generation

## 🛠️ Technologies Used

- **Framework:** Next.js 15 with App Router
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Internationalization:** next-intl
- **Form Handling:** React Hook Form
- **Icons:** Lucide React
- **Deployment:** Vercel
- **Font Optimization:** Next.js Font optimization

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/LauraBLei/fmkgulve.git
cd fmkgulve
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
npm run lint     # Run ESLint
```
