## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+

### Installation

1. Clone the repository:

```bash
git clone https://github.com/LauraBLei/fmkgulve.git
cd fmkgulve
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```
